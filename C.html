C Language Header Files Guide

stdio.h

Input/output functions like printf(), scanf(), fopen().

stdlib.h

Memory allocation, random numbers, conversions (malloc, rand).

string.h

String handling functions like strcpy(), strlen().

math.h

Math functions like sqrt(), pow(), sin().

time.h

Date and time functions like time(), clock().

ctype.h

Character testing like isdigit(), isalpha().

assert.h

Debugging using assert().

limits.h

Defines limits of data types (INT_MAX, CHAR_MIN).

float.h

Floating-point limits like FLT_MAX.

signal.h

Handles signals like interrupts.

setjmp.h

Jump control (setjmp, longjmp).

stdarg.h

Functions with variable arguments (va_list).

errno.h

Error handling (errno variable).

locale.h

Localization settings.

wchar.h

Wide character handling.

conio.h

Console input/output (non-standard, Turbo C).

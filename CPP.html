C++ STL Header Files Guide

iostream

Input/output streams like cout, cin.

vector

Dynamic array container.

string

String class for text handling.

map

Key-value pairs (ordered).

set

Unique elements (ordered).

unordered_map

Key-value pairs (hash-based, fast).

unordered_set

Unique elements (hash-based).

algorithm

Sorting, searching (sort, binary_search).

cmath

Math functions like sqrt(), pow().

iomanip

Output formatting (setprecision, setw).

fstream

File handling (read/write files).

sstream

String stream processing.

thread

Multithreading support.

mutex

Thread synchronization.

chrono

Time utilities and measurements.

memory

Smart pointers (shared_ptr, unique_ptr).

utility

Pair, swap, and helpers.

bitset

Fixed-size bit manipulation.

deque

Double-ended queue.

list

Doubly linked list container.
